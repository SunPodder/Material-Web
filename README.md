# Material You Web

Applies Material You theme to your everyday websites. This is a work in progress, and not all websites are supported yet.

## Supported Websites

- [ ] Github (WIP)
- [ ] Youtube
- [ ] Reddit
- [ ] Facebook
- [ ] Whatsapp

## Installation

Build the project using Bun and load the unpacked extension in Firefox (Chrome support is coming soon).
```
bun install
bun build
```

I plan to add matugen based theme generation in the future, so that you can generate themes based on your wallpaper. For now, you can edit the `colors.css` file to change the theme colors.
